# Micro-Dot-RPG

A RPG system "that would fit on a stamp" by TrapperQ

```
Micro-Dot-RPG
Nobody dies
State action and 
Roll 1d4
1. Amazing fumble
2. Fail (Unless part Of PC Occupation)
3. Success
4. Amazing Critical
```
